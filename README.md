# LeetCode Problem Rating Displayer

该脚本是一个油猴脚本，作用是在Leetcode题目页面显示出该题目的难度值，题目难度文件来源于: `https://zerotrac.github.io/leetcode_problem_rating/data.json`

自用阶段，满足个人需求，看情况更新

## 效果

**Version 0.3.0**
